# 2048 Web Game

经典2048数字合并游戏 - 使用HTML、CSS和JavaScript开发的网页版本。

## 游戏玩法

- 使用 **方向键** 或 **WASD** 键移动方块
- 在移动设备上，**滑动屏幕** 即可移动方块
- 相同数字的方块相撞时会合并成更大的数字
- 目标：合成 **2048**！

## 功能特点

- ✅ 完整的2048游戏逻辑
- ✅ 分数和最高分系统
- ✅ 流畅的动画效果
- ✅ 响应式设计，适配桌面和移动设备
- ✅ 触摸滑动支持
- ✅ 本地存储最高分

## 技术栈

- HTML5
- CSS3 (动画、响应式布局)
- Vanilla JavaScript

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
